# Square UI

Square UI is a lightweight UI theme for Firefox-based browsers.

It gives the browser a cleaner, sharper look by reducing rounded corners across the interface and page content.

## Highlights

- Minimal, square visual style
- Separate controls for browser UI and website content
- Works best on Zen, with support for other Firefox forks

## Installation

Install from the marketplace, or add this repository as a local source.

After installing, enable the mod and choose the options you want.

## Browser support

Zen is the primary target. Other Firefox-based browsers are supported where possible.
